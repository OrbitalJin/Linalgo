# LinAlGo

Linear algebra library for Go, written in Go

## Features

-   Matrices

*   [x] Addition/Substraction
*   [x] Scalar/Transformation
*   [x] Transpose
*   [x] Negate
*   [x] Value Setting
*   [x] Equality Checking
*   [x] Transposing
*   [ ] Identity
*   [ ] Dot Product
*   [ ] Division (A/B = A \* B^-1)
*   [x] Add tests
*   Linear Systems
